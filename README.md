# 원티드 AI 면접 코칭 클론

AI 기반 면접 코칭 기능을 복제한 프로젝트로, 예상 면접 질문 제공과 답변에 대한 상세 피드백을 제공합니다.

---

## 주요 기능

- 직무와 업계를 기반으로 면접 질문을 예측.
- 사용자의 답변을 분석하고 실질적인 피드백 제공.

---

## 기술 스택

### Front

- **React**
- **TypeScript**
- **Zustand**
- **React Query**
- **Tailwind CSS**
- **Vite**

### 테스트

- **Jest**
